# Do Not Waste Your Money

> **A simple web application built to serve as a reality check for a friend, helping them stop wasting money on unrequited love.**

This project was created as a fun, intervention-style tool to help a friend "disembark" from being overly infatuated and to prevent them from making irrational financial decisions for a crush. It was built with the assistance of Google Gemini.

---

## Purpose

* **Reality Check:** Reminds users to think twice before opening their wallets for someone who might not feel the same way.
* **Financial Protection:** Aims to save money that could be better spent on oneself or more meaningful things.
* **Friendly Intervention:** Developed as a caring, humorous, and straightforward wake-up call for a friend in need.

---

## Tech Stack

This is a lightweight static web application built with fundamental front-end technologies:
* **HTML:** Page structure and content.
* **CSS:** Styling and visual layout.
* **JavaScript:** Interactive elements and core logic.
